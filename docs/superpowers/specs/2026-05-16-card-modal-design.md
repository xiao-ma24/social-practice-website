# 调研区域 & 非遗项目 "概览 + 弹窗" 双层展示

**日期**: 2026-05-16
**状态**: 已确认，待实施

---

## 目标

将第三板块（实践内容）中调研区域卡片和非遗项目卡片改为"概览 + Modal 弹窗"双层展示模式，减少页面信息密度，保持高级清爽的杂志感。

---

## 范围

| 板块 | 改动 |
|------|------|
| 调研区域卡片 (`surveyAreas`) | 概览卡片 + 点击弹窗 |
| 非遗项目卡片 (`heritageItems`) | 概览卡片 + 点击弹窗 |
| 实践记录照片墙 (`practicePhotos`) | 不变 |
| 城市大图、标题区、感悟区、信息条 | 不变 |
| 左侧导航栏 | 不变 |
| 省份六宫格交互 | 不变 |

---

## 第一层 · 概览卡片

### 调研区域卡片

- 深色背景 + 渐变遮罩 + 白色文字（保持现有样式）
- 图片 + 名称 + 一句描述（现有 `description` 字段，约20字以内）
- 右下角加 **小放大镜图标** 暗示可点击

### 非遗项目卡片

- 浅色圆角卡片 + 图片 + 名称 + 级别标签（保持现有样式）
- 描述文字改为简短版本（新增 `shortDesc` 字段，30~50字）
- 右上角或底部加 **小箭头/查看图标** 暗示可点击

### 交互

- hover: 微微上浮 + 阴影增强 + 出现"点击查看"提示
- 点击: 打开 Modal
- cursor: pointer

## 第二层 · Modal 弹窗

### 布局（从上到下）

```
┌─────────────────────────────────────┐
│  ✕ 关闭按钮（右上角，圆形，hover变色） │
├─────────────────────────────────────┤
│                                     │
│        大图（宽度撑满，自适应）        │
│                                     │
├─────────────────────────────────────┤
│  名称（大号，深蓝色，宋体）            │
│  类别标签（非遗级别 / 调研区域）       │
├─────────────────────────────────────┤
│  📖 介绍                            │
│  （段落文字，行高宽松）               │
│                                     │
│  🔍 调研过程                         │
│  （段落文字）                        │
│                                     │
│  💭 实践感受                         │
│  （段落文字，可加引用风格）            │
└─────────────────────────────────────┘
```

### 样式规范

- 背景色: 米白色 `#FAF8F5`，与页面整体协调
- 圆角: `16px`
- 最大宽度: `680px`
- 遮罩背景: `rgba(15, 43, 92, 0.4)` 深蓝半透明
- 标题: 宋体，深蓝 `#0F2B5C`
- 正文: 系统字体，深灰 `#5C5C5C`
- 分割线: 淡金色细线
- 淡入动画: `opacity 0→1 + translateY(20px→0)`，`0.3s ease`
- 关闭按钮: 圆形，hover 变金色

### 交互

| 操作 | 行为 |
|------|------|
| 点击概览卡片 | Modal 淡入弹出 |
| 点击 ✕ 按钮 | Modal 淡出关闭 |
| 点击遮罩区域 | Modal 淡出关闭 |
| 按 ESC 键 | Modal 淡出关闭 |
| Modal 打开时 | body 滚动锁定 (`overflow: hidden`) |
| 点击 Modal 内部 | 不关闭（阻止冒泡） |
